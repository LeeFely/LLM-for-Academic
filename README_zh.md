# AcademicGPT
A Prompt Engineering for scientific freshman to get started quickly
本项目由非CS专业科研人员制作，专注于对本地现有文献进行训练，帮助实验室新人/刚刚步入科研领域的大学生快速入门导师所在领域，个性化的学习AI助手，因为涉及版权问题，故后续会考虑对SCI一区期刊（自然科学类期刊）的unopen access文章的摘要和open access的全文进行训练，本项目严禁用于论文写作，后果自负！
本项目由现有开源LLM进行训练，接入知识图谱，对文献内容进行可视化。项目宗旨是个性化知识服务，不对全网进行训练，故可实现本地化部署。
注：初代模型预计将仅对文本就行训练，后续可能开放多模态。
项目作者是一位来自Bioscience本科生，但是项目将在一年后，即我考研结束后开始，即2023/06开始。因本人能力有限，希望有CS专业背景的开发者与我交流。
